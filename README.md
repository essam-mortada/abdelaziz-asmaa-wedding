# Abdelaziz & Asmaa Wedding Invitation

A responsive, bilingual wedding invitation website for Friday, 7 August 2026.

## Preview

Open `index.html` directly, or serve the folder with any static web server.

## Included details

- Couple: Abdelaziz & Asmaa / عبد العزيز وأسماء
- Date: Friday, 7 August 2026
- Time: 7:00 PM (Cairo time)
- Venue: Al Gawhara Hall, Infantry Officers Club / دار ضباط المشاة — قاعة الجوهرة

The site includes a full-viewport wedding envelope and wax-seal opening interaction, Three.js rose petals, intertwined wedding rings, a choreographed hero reveal, countdown, bilingual details, directions, calendar download, invitation gallery, responsive layouts, keyboard focus states, and reduced-motion support.

Three.js r184 is bundled locally in `vendor/` with its MIT license, so the background effects do not require a CDN.
